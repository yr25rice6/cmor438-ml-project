algorithms covered in the course. Each algorithm has its own numbered
subdirectory so the material can be browsed in the order it is taught.

The subdirectories are placeholders for now; the dedicated notebooks will be
added in a later step. Each folder currently holds a `.gitkeep` placeholder so
the directory is tracked in git.
